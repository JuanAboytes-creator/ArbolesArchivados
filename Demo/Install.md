# Instalación y Uso

## Requisitos
- g++ (compilador C++17)
- Git (para clonar)
- Linux/macOS (probado en Ubuntu)

## Instalación Rápida
```bash
# Clonar repositorio
git clone https://github.com/TU-USUARIO/ArbolesArchivados.git
cd ArbolesArchivados

# Compilar
cd src
make

# Ejecutar
./arboles_archivados
